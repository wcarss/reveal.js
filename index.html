Crash Course in Vim

oh god what's happening

Vim has two modes: command and edit.

It starts in command mode.

You can't type stuff normally in command mode.

Get to edit (aka insert) mode

You can type freely in edit mode.

To get to edit mode, hit 'i'.

You can't save or quit in edit mode.

Get to command mode

You can save and quit in command mode.

Hitting <Escape> will get you back to command mode.

saving

type ':' to begin a command.

type 'w' and then hit <Enter> to write (or save) the file.

quittng

type ':' to begin another command.

type 'q' and then hit <Enter> to quit the editor.

deal with it

Q: Why did we just do this?

A: git opens vim by default when we make a commit.

Gotta learn sometime, and it isn't that bad.

Quick Summary

edit mode     # write words here
command mode  # write commands here
i             # command mode -> edit mode
<ESC>         # edit mode -> command mode
:w            # (command only) saves the file
:q            # (command only) quits the file
